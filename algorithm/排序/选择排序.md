## 思路

- 第一轮,找到最小的元素,和数组第一个数交换位置. 
- 第二轮,找到第二小的元素,和数组第二个数交换位置...
- 直到最后一个元素,排序完成. 

```js
function select(arr) {
  if (Array.isArray(arr)) {
    for (let i=0; i< arr.length - 1; i++) {

      let min = i
      for (let j=i+1; j<arr.length; j++) {
        if (arr[j] < arr[min]) {
          //  记录序号即可,不需要交换
          min = j
        }
      }

      [arr[i], arr[min]] = [arr[min], arr[i]]
    }
    return arr
  }

  return []
}


```
