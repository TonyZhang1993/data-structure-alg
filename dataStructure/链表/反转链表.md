## 反转链表
输入一个链表，反转链表后，输出新链表的表头。

```js
let reverseList = (head) => {
  //  存储当前节点
  let currentNode = null
  //  新链表表头
  let headNode = head

  while (head && head.next) {
    currentNode = head.next
    head.next = currentNode.next
    currentNode.next = head

    headNode = currentNode
  }

  return headNode
}
```
